### Storybook
A Storybook server is provided to allow browsing the components used in this project.  
```
npm run storybook
```

### Testing
Run Jest and React-Testing-Library tests on React code.
```
npm run test
```

Run server-side API testing code.
```
cd server
npm run test
```
